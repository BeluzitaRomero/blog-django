Mi nombre es Belen Romero y soy desarrolladora fullstack 😄

En este momento me encuentro culminando mi cursada de desarrollo backend con Python y Django el cual me permitio desarrollar este proyecto fullstack... ¡desde cero! ¡Te agradezco por estar aquí viendo mi trabajo! Siempre es bueno tener un feedback de nuestros trabajos 👏

Este proyecto es un Blog, el cual se puede navegar para leer los post de usuarios, pero para poder realizar un post personal, debes registrarte para ello! Cuanta con registro y login de usuarios, los cuales son autenticados por django.

Una vez que el usuario esta creado, podes ingresar y crear tus propios post con titulos, subtitulos, contenido e imagenes. Una vez hecho, el resto de usuarios registrados podran ver tu post y dejarte su comentario. Asimismo, tu tambien puedes comentar los post de otros y... no tengas problema! Si te arrepientes, podes borrar tus comentarios sin problema! 🤭
Tus post tambien podras eliminarlos y actualizarlos si lo necesitas.

Tambien cuentas con una seccion para editar tu perfil, sumar nombre y apellido y una imagen para tu avatar.
